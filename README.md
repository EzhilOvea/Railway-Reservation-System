# Railway Reservation System 🚆

A desktop-based Railway Reservation System developed using Python and Tkinter. This application allows users to book train tickets through an interactive graphical interface and simulates a real-world booking system.

---

## 🔧 Features
- User login authentication
- View available trains
- Enter travel details (source, destination, date)
- Train selection system
- Ticket booking functionality
- Multiple payment options:
  - UPI
  - Net Banking
  - Credit Card
  - Debit Card
- Ticket generation with complete details

---

## 🛠️ Technologies Used
- Python
- Tkinter (GUI)
- MySQL (Database)

---

## 📌 Project Description
This project simulates a railway reservation system where users can log in, search for trains, and book tickets. The application uses a MySQL database to store user credentials and train details. It provides a step-by-step booking process including travel input, train selection, payment, and ticket generation.
